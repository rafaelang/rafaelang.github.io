tags: python django middlewares exceptions tips

Essa integração foi desenvolvida com o Django 2.0. E com o modulo better exceptions

  • Instale o better exceptions
  • Adicione esse arquivo e configuração ao seu projeto Django.

Arquivo: myproject/middlewares.py

import sys
from better_exceptions import format_exception
from django.utils.deprecation import MiddlewareMixin


class BetterExceptionMiddleware(MiddlewareMixin):
    def process_exception(self, request, exception):
        print(format_exception(*sys.exc_info()))

Edite o arquivo settings de desenvolvimento do seu projeto Django

# Additional middleware
MIDDLEWARE += [
    'myproject.middlewares.BetterExceptionMiddleware',
    ]

É isso. Só se divertir debugando suas exceptions com mais informações.
