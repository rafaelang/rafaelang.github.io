  • Validar HTML com a biblioteca padrão do python
>>> from xml.etree import ElementTree
>>> ElementTree.fromstring('<br/>') #Ok
>>> ElementTree.fromstring('<p>Teste & teste</p>') #xml.etree.ElementTree.ParseError: not well-formed (invalid token): line 1, column 10
  • Corrigindo rapidamente com BeautifulSoap
>>> from bs4 import BeautifulSoap
>>> ElementTree.fromstring(str(BeautifulSoup('<p> teste & teste</p>', 'lxml'))) #Ok
